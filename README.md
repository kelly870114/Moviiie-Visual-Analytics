# Moviiie-Visual-Analytics
In this project, we use React.js with Flask, along with Tableau API.
## Set up Environment
1. Install Flask
```
pip3 install -r requirements.txt
```
2. Install packages under `/moviiie-visual-analytics/dashboard`
```
cd dashboard
npm install
```

## Run the Environment 
1. Run Flask under `/moviiie-visual-analytics/server`
```
cd server
flask run 
```

2. Run React App under `/moviiie-visual-analytics/dashboard`
```
cd dashboard
npm run start
```
