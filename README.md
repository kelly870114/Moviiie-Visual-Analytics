# Moviiie-Visual-Analytics
In this project, we use React.js with Flask, along with Tableau API.
## Set up Environment
1. Install Flask
```
pip3 install -r requirements.txt
```
2. Install packages
```
cd dashboard
npm install
```

## Run the Environment
1. Run Flask
```
cd server
flask run 
```

2. Run React App
```
cd dashboard
npm run start
```
